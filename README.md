# Yenasoy
Industrial automation services company website based in Uzbekistan.

## About

Yenasoy provides comprehensive automation solutions for manufacturing production lines, specializing in:

- **PLC Programming** - Control logic design and implementation
- **HMI Development** - Operator interfaces and visualization systems
- **SCADA & VFD Integration** - Supervisory control and variable frequency drives
- **Diagnostics & Support** - Remote and on-site troubleshooting

### Services

1. **Fast/Reactive Support** - Emergency diagnostics, remote/on-site recovery
2. **Planned Engineering** - PLC/HMI modifications and system upgrades
3. **New Development** - Complete automation systems from scratch
4. **Documentation & Training** - System handover and staff training

### System Complexity Classes

| Class | Size | Motors/VFDs | Sensors |
|-------|------|-------------|---------|
| S (Simple) | Up to 30m | ≤5 | ≤15 |
| M (Medium) | 30-100m | 6-15 | 16-50 |
| L (Complex) | 100-200m | 16-40 | 50-120 |
| XL (High-Complexity) | 200m+ | 40+ | 120+ |
